# Complete Google Sheet Structure

Your Google Sheet should have multiple sheets (tabs) for different data:

## Sheet 1: "Deals"
Contains your pipeline deals

| Deal Name | Amount | Stage | Close Date | Probability | Created Date |
|-----------|--------|-------|------------|-------------|--------------|

## Sheet 2: "Metrics"
Contains all your SaaS metrics (2 columns: Metric Name, Value)

| Metric | Value |
|--------|-------|
| ARR Target H2 2025 | 675000 |
| Current ARR | 2513000 |
| MRR | 209417 |
| Gross Renewal Rate | 92.5 |
| Net Revenue Retention | 122 |
| Burn Multiple | 1.2 |
| CAC Payback Months | 9 |
| LTV to CAC Ratio | 4.2 |
| Logo Churn Rate | 2.8 |
| Magic Number | 1.3 |
| Cash Runway Months | 18 |
| ARR Growth Rate YoY | 22 |

| Quarter | New Customer | Existing Customer | Total |

| Q1 2024 | 90000 | 54000 | 144000 |
